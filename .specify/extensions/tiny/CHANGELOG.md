# Changelog

## 1.0.0 (2026-04-10)

- Initial release
- Add `/speckit.tiny.specify` command for generating single-file lightweight specs
- Add `/speckit.tiny.implement` command for implementing directly from tinyspec files
- Add `/speckit.tiny.classify` command for routing tasks to tinyspec or full SDD
- Optional `before_specify` hook for auto-classifying task complexity
- Addresses community request in issue #1174 (22+ reactions)
